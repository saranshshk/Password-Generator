#  Password Generator

A modern and responsive password generator built with **React** that helps users create strong and secure passwords.

##  Features
- Generate secure passwords instantly
- Customize length (6–30 characters)
- Include uppercase, lowercase, numbers & symbols
- Real-time strength indicator (Weak / Medium / Strong)
- One-click copy to clipboard
- Clean and user-friendly UI

##  Tech Stack
- React.js
- JavaScript
- Tailwind CSS
- React Toastify

##  Live Demo

##  Installation

git clone https://github.com/saranshshk/Password-Generator.git
cd Password-Generator-Main
npm install
npm run dev
